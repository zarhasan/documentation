== Description ==
Documentation theme designed for creating product documentation, release notes, and changelogs. It features a built-in search engine to enhance navigation, making it ideal for structured and organized content presentation. The theme is fully responsive, SEO-friendly, and Gutenberg-ready.

== Features ==
* Built-in search engine
* Optimized for documentation websites
* Supports release notes and changelog pages
* Responsive and mobile-friendly design
* SEO-optimized
* Gutenberg-compatible
* Easy installation and setup

== Installation ==
1. Upload the theme to the `/wp-content/themes/` directory or install it via the WordPress theme installer.
2. Activate the theme through the 'Appearance > Themes' menu in WordPress.
3. Configure theme options as needed in the WordPress Customizer.
